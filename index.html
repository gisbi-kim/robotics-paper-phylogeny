🧬 4-depth 분류 용어

생물 계통도(phylogenetic taxonomy)에서 빌려온 4단계. 위에서 아래로 점점 좁아지는 카테고리.
L1
Phylum
13개
대분류 — 로봇공학 안의 큰 분야 예: Manipulation, Locomotion, SLAM & Localization
L2
Class
~100개
중분류 — Phylum 안의 주요 갈래 예: Manipulation ▸ Grasping
L3
Order
~330개
소분류 — Class 안의 세부 주제 예: Grasping ▸ Grasp Planning / Synthesis
L4
Genus
가변
세부 — Order 안의 구체적 접근법. top 45 Order에만 sub-rule이 있고 나머지는 모두 (general). 예: Grasp Planning ▸ Learning-based Grasping
왜 생물 계통도? 같은 주제도 논문마다 표현이 다르다 (예: "laser place recognition" ≈ "point cloud loop detection"). 단순 키워드/TF-IDF 대신 시맨틱 동의어 클러스터로 묶고, 분야 → 갈래 → 주제 → 접근법으로 내려가는 트리에 매핑. 결과적으로 "어떤 분야가 언제 등장/소멸했는지" "한 주제 안에 어떤 패러다임이 공존하는지"를 한눈에 본다.

  • 📊 Sim-to-Real Legged: 2018년 첫 등장, 2021-2025에 누적 18편 — 단 4년만에 표준 패러다임화.

🥧 Radial Tree (D3.js)

생물 계통도 파이 차트. 사이드바 검색으로 단어가 들어간 wedge들을 한꺼번에 하이라이트. 호버로 강조, 클릭으로 계열 선택 후 ←→↑↓로 탐색.

🖱️ Open in new tab (full screen)
📍 사용법 + 추천 탐색
  • 🔍 검색 → 단어를 입력하거나 추천 칩(↻로 갱신)을 클릭. 매칭된 wedge에 주황 glow + 사이드바에 결과 리스트.
  • 결과 클릭 → 해당 wedge로 점프, 계열(조상+후손) lineage 하이라이트.
  • 호버 → wedge 두꺼워짐 + 사이드바 HOVER 카드.
  • ←/→ = 형제, ↑ = 부모, ↓ = 첫 자식 (가장 큰 child부터)
  • 📊 추천 탐색: Manipulation → Contact-rich → Deformable Object의 Genus까지 들어가보면 Cloth/Garment vs Rope/Cable vs Liquid 비중 비교 가능

🌳 Horizontal Collapsible Tree (D3.js)

왼쪽 root에서 오른쪽으로 가지치는 전형적 phylogenetic tree. 노드 또는 라벨 클릭으로 펼치기/접기. 기본은 Phylum까지만 펼쳐져 있고 더 깊이 들어가려면 클릭하거나 사이드바 버튼 사용.

🖱️ Open in new tab (full screen)
📍 사용법
  • 노드/라벨 클릭 → 자식 펼치기/접기 + focus 선택 (해당 lineage 강조, 나머지 dim)
  • 호버 → 사이드바 HOVER 카드에 정보 표시 (실시간)
  • ↑/↓ = 같은 부모의 형제 노드로 이동, ← = 접기, → = 펼치기, Esc = 해제
  • 사이드바 버튼: L2까지 = 모든 Class 펼침, L3까지 = Order까지, 전부 접기 = Phylum만
  • 파란 외곽 동그라미가 있는 노드 = 접힌 자식이 있음을 의미
